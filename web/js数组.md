# JavaScript 数组常用方法表

| 方法              | 作用                   | 使用案例                               | 输出                                 | 使用场景                          |
| ----------------- | ---------------------- | -------------------------------------- | ------------------------------------ | --------------------------------- |
| `push()`          | 末尾添加元素           | `arr = [1,2]; arr.push(3)`             | `arr = [1,2,3]` ｜ 返回新长度 `3`    | 追加列表数据                      |
| `pop()`           | 删除最后一个元素       | `arr = [1,2,3]; arr.pop()`             | `arr = [1,2]` ｜ 返回 `3`            | 栈结构出栈、删除最后一项          |
| `unshift()`       | 开头添加元素           | `arr = [2,3]; arr.unshift(1)`          | `arr = [1,2,3]` ｜ 返回新长度 `3`    | 头部插入数据                      |
| `shift()`         | 删除第一个元素         | `arr = [1,2,3]; arr.shift()`           | `arr = [2,3]` ｜ 返回 `1`            | 队列出队                          |
| `splice()`        | 增删改数组             | `arr = [1,2,3]; arr.splice(1,1,99)`    | `arr = [1,99,3]` ｜ 返回 `[2]`       | 删除、插入、替换中间元素          |
| `slice()`         | 截取数组               | `[1,2,3,4].slice(1,3)`                 | `[2,3]` ｜ 不改原数组                | 分页、截取部分数据                |
| `concat()`        | 合并数组               | `[1,2].concat([3,4])`                  | `[1,2,3,4]`                          | 合并多个列表                      |
| `join()`          | 数组转字符串           | `['a','b','c'].join('-')`              | `'a-b-c'`                            | 拼接展示文本                      |
| `indexOf()`       | 查找元素下标           | `[1,2,3].indexOf(2)`                   | `1` ｜ 找不到返回 `-1`               | 查找基本类型位置                  |
| `includes()`      | 判断是否包含元素       | `[1,2,3].includes(2)`                  | `true`                               | 判断值是否存在                    |
| `find()`          | 找第一个满足条件的元素 | `[1,2,3].find(x => x > 1)`             | `2` ｜ 找不到返回 `undefined`        | 查找对象或条件匹配项              |
| `findIndex()`     | 找第一个满足条件的下标 | `[1,2,3].findIndex(x => x > 1)`        | `1` ｜ 找不到返回 `-1`               | 找到位置后修改或删除              |
| `forEach()`       | 遍历数组               | `[1,2,3].forEach(x => console.log(x))` | 依次输出 `1 2 3` ｜ 返回 `undefined` | 单纯遍历、打印、执行副作用        |
| `map()`           | 映射生成新数组         | `[1,2,3].map(x => x * 2)`              | `[2,4,6]`                            | 数据转换、React 列表渲染          |
| `filter()`        | 过滤数组               | `[1,2,3,4].filter(x => x > 2)`         | `[3,4]`                              | 筛选符合条件的数据                |
| `some()`          | 是否有一个满足条件     | `[1,2,3].some(x => x > 2)`             | `true`                               | 判断是否存在符合条件的数据        |
| `every()`         | 是否全部满足条件       | `[1,2,3].every(x => x > 0)`            | `true`                               | 表单校验、权限校验                |
| `reduce()`        | 累计计算               | `[1,2,3].reduce((a,b) => a + b, 0)`    | `6`                                  | 求和、统计、分组、数组转对象      |
| `sort()`          | 排序                   | `[3,1,2].sort((a,b) => a - b)`         | `[1,2,3]` ｜ 改原数组                | 数字排序、列表排序                |
| `reverse()`       | 反转数组               | `[1,2,3].reverse()`                    | `[3,2,1]` ｜ 改原数组                | 倒序展示                          |
| `flat()`          | 扁平化数组             | `[1,[2,[3]]].flat(2)`                  | `[1,2,3]`                            | 处理嵌套数组                      |
| `Array.isArray()` | 判断是否为数组         | `Array.isArray([1,2])`                 | `true`                               | 类型判断                          |
| `Array.from()`    | 类数组转数组           | `Array.from('abc')`                    | `['a','b','c']`                      | 字符串、Set、伪数组转数组         |
| `Array.of()`      | 创建数组               | `Array.of(1,2,3)`                      | `[1,2,3]`                            | 创建数组，避免 `new Array()` 歧义 |
| `fill()`          | 填充数组               | `new Array(3).fill(0)`                 | `[0,0,0]` ｜ 改原数组                | 初始化数组                        |
| `toSorted()`      | 返回排序后的新数组     | `[3,1,2].toSorted((a,b) => a - b)`     | `[1,2,3]` ｜ 不改原数组              | 不想影响原数组时排序              |
| `toReversed()`    | 返回反转后的新数组     | `[1,2,3].toReversed()`                 | `[3,2,1]` ｜ 不改原数组              | 不想影响原数组时倒序              |

# 高频易混点

| 对比                    | 区别                                      | 示例输出                                                     |
| ----------------------- | ----------------------------------------- | ------------------------------------------------------------ |
| `map()` vs `forEach()`  | `map` 返回新数组；`forEach` 只遍历        | `[1,2,3].map(x=>x*2)` → `[2,4,6]` ｜ `forEach` → `undefined` |
| `slice()` vs `splice()` | `slice` 不改原数组；`splice` 会改原数组   | `slice(1,3)` → `[2,3]` ｜ `splice(1,2)` → 删除 `[2,3]`       |
| `find()` vs `filter()`  | `find` 返回第一个元素；`filter` 返回数组  | `find(x=>x>2)` → `3` ｜ `filter(x=>x>2)` → `[3,4]`           |
| `some()` vs `every()`   | `some` 有一个满足即可；`every` 要全部满足 | `[1,2,3].some(x=>x>2)` → `true` ｜ `every(x=>x>2)` → `false` |
| `sort()` 默认排序       | 默认按字符串排序，不是数字大小            | `[1,10,2].sort()` → `[1,10,2]`                               |
| `sort()` 数字升序       | 需要比较函数                              | `[1,10,2].sort((a,b)=>a-b)` → `[1,2,10]`                     |

# 是否修改原数组

| 会修改原数组   | 不修改原数组   |
| -------------- | -------------- |
| `push()`       | `map()`        |
| `pop()`        | `filter()`     |
| `unshift()`    | `slice()`      |
| `shift()`      | `concat()`     |
| `splice()`     | `join()`       |
| `sort()`       | `find()`       |
| `reverse()`    | `findIndex()`  |
| `fill()`       | `includes()`   |
| `copyWithin()` | `some()`       |
|                | `every()`      |
|                | `reduce()`     |
|                | `flat()`       |
|                | `toSorted()`   |
|                | `toReversed()` |

面试重点记这几个：`map`、`filter`、`reduce`、`forEach`、`find`、`some`、`every`、`slice`、`splice`、`sort`。
